How It Works

EC2 Restart Manager retrieves instance data from an Amazon S3 bucket. The S3 bucket is updated every 30 minutes by a separate background process.

Use the Update button to manually fetch the latest instance information from the S3 bucket at any time. This ensures you always have access to the most current data when needed.

The Status page displays the outcome of the last restart operation, providing transparency and clarity about recent actions.

Permissions and Access

Everyone can view instance information, including filtering by owner, service, AWS account name, or region.

Restart operations are restricted to members of the Active Directory (AD) group SG-APP-EC2-restart-manager. This ensures that only authorized users can perform critical actions.
